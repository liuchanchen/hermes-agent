# H3C 超节点产品规格

## S80000 液冷整机柜

### 概览
- 定位：面向万亿级参数模型训推场景
- 设计理念：Rack as a Computer，以网强算
- 形态：整机柜超节点

### 物理规格

| 参数 | 数值 |
|------|------|
| 机柜尺寸 | 600mm × 1200mm × 2300mm |
| 可用槽位 | 46SU |
| 计算节点 | 最多16台 S8400 G7 |
| 交换节点 | 最多8个 |
| GPU卡数 | 64卡（单柜）/ 256卡（4柜集群，2层无阻塞交换） |
| 空柜重量 | 350kg |
| 满配重量 | 约1400kg（含包材150kg） |

### 机柜空间分布（46SU）
| SU范围 | 用途 |
|--------|------|
| 1-2 | 假面板 |
| 3-4, 39-40 | Power Shelf（电源模块+RMC） |
| 6-7, 10-13, 16-17, 26-27, 30-33, 36-37 | 计算节点 |
| 8-9, 14-15, 28-29, 34-35 | 交换节点 |
| 18-19, 24-25 | 毛刷+理线架 |
| 41-46 | TOR交换机(3SU) + 理线架(3SU) |

### 供电系统
| 参数 | 规格 |
|------|------|
| 供电方案 | Busbar集中供电 |
| 最大供电能力 | 120kW |
| 输入制式 | 380V AC三相五线 |
| 连接器 | IEC 60309-2 工业连接器，63A |
| 每计算节点 | ≥8kW/槽位 |
| 每交换节点 | ≥3kW/槽位 |
| 冗余 | 11+1 PSU冗余 |
| 电压范围 | 198-264V AC |

### 散热系统
| 参数 | 规格 |
|------|------|
| 散热方式 | 冷板液冷为主，风冷辅助 |
| 冷却液 | 去离子水、PG25丙二醇、EG25乙二醇 |
| 进液温度 | 5-40°C |
| 整柜流量 | ≥96 L/min（CPU+GPU液冷）/ ≥112 L/min（CPU+GPU+DIMM液冷） |
| 工作压力 | ≤3.0 Bar（推荐≤2.5） |
| 整柜压差 | ≥60 Kpa |
| CPU冷板 | 钎焊工艺 |
| 快接头型号 | UQDB06（可适配其他型号） |

### 液冷管路
- Manifold顶部自动排气阀
- 泄漏液体通过软管导入排液管、喷溅盒
- 节点鹰嘴泄漏液体汇入机柜底部积液盘
- 冷板模组集成漏液检测绳
- Manifold连接漏液检测绳

### 环境参数
| 参数 | 工作 | 贮存 |
|------|------|------|
| 温度 | 5°C-40°C | -40°C-70°C |
| 湿度 | 8%-90%（无冷凝） | 5%-95%（无冷凝） |
| 海拔 | -60m-3000m（>1500m每100m降0.33°C） | -60m-3000m |
| 运输坡度 | ≤10度（建议6度） | - |

### 关键特性
- **高性能**：推理性能提升100%+，单token成本降低50%+
- **高可靠**：Cable tray互联可靠性提升10倍，PD实例自动切换
- **极简运维**：水电网络三总线盲插，统一运维界面
- **高效散热**：液冷覆盖CPU/GPU/DIMM
- **一体化交付**：工厂预集成，整柜运输，即插即用
- **管理**：RMC机柜级管理，状态监控/日志/报警/资产管理

## 软件栈

### 分层架构
1. **基础资源层**
   - NingOS定制操作系统（NUMA亲和、内存Pin驻留）
   - GPU驱动、HDM/BIOS/RMC设备管理
   - Kubernetes容器编排

2. **加速计算与通信层**
   - DNN算子库：多粒度并行、矩阵乘加速、算子融合
   - CCL集合通信库：拓扑自动探测、最优路径选择、Primary-Backup QP
   - Runtime运行库与SDK

3. **框架与平台层**
   - 训练：Megatron, DeepSpeed, PyTorch
   - 推理：vLLM, SGLang（PD分离、统一KV Cache）
   - 业务使能平台

4. **智能管控与运维层**
   - SeerEngine-DC：网络/计算/存储自动化配置
   - SeerAnalyzer-DC：算网存全域运维分析、智能故障推理
   - UC-IOM：基础设施监控
   - UniSystem：批量部署、固件升级、生命周期管理
