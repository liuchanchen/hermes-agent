# 超节点部署要求清单

## 机房基础设施要求

### 供电系统
| 项目 | 要求 |
|------|------|
| 输入制式 | 380V AC 三相五线制 |
| 连接器类型 | IEC 60309-2 工业连接器（3P+N+E），额定63A |
| 供电架构 | 双路AC+AC输入，每PowerShelf独立供电 |
| 电压范围 | 198-264V AC，波形失真≤5% |
| 配电方式 | 列头柜或小母线，预留扩容空间 |
| 断路器 | C型或D型热磁脱扣，Icu≥15kA，Ics≥50%Icu |
| 备用空间 | 桥架至少60%备用空间 |
| 电源线 | 截面积≥16mm²，工作温度5-40°C |
| 强电/弱电间距 | ≥200mm（受限≥150mm） |

### 接地与安全
| 项目 | 要求 |
|------|------|
| 接地电阻 | ≤1Ω |
| 接地点 | 机柜顶部/底部前后M6或M8 |
| 接地线截面 | ≥16mm² |
| 绝缘电阻 | ≥10MΩ（500V兆欧表） |
| 绝缘强度 | 1890V/50Hz，1min无击穿，漏电流≤10mA |
| 防雷 | 浪涌保护器4+0共模保护，远程监控 |
| 接地系统 | TNS系统 |

### 散热系统
| 项目 | 要求 |
|------|------|
| 散热方式 | 冷板液冷（CPU+GPU+可选DIMM） |
| 冷却液 | 去离子水+缓释剂或20-25%乙二醇/丙二醇 |
| 进液温度 | 25-40°C，>露点温度+3°C |
| 管路接口 | DN50，ISO2852标准法兰 |
| 二次管路材质 | 304及以上不锈钢预制硬管 |
| 压损要求 | 并联环路压损差≤15% |
| 接管高度 | ≥400mm（DN50+软管弯曲半径） |
| 阀门类型 | 自封式互锁型球阀 |
| 漏液检测 | 1米级定位漏水告警绳，托盘覆盖 |
| 排水管 | 外径12-20mm，长度1500mm |

### 机柜与承重
| 项目 | 要求 |
|------|------|
| 楼板活荷载 | ≥16kN/m² |
| 机柜底座 | 镀锌钢板焊接专用底座 |

### 运输要求
| 项目 | 要求 |
|------|------|
| 运输车辆 | 公路型精密仪器气垫减震货车 |

## 超节点选型决策框架

### 适用性判断
适合超节点的场景（通信占比>20-30%）：
- 万亿参数以上模型训练
- 实时推理服务（低延迟要求）
- 内存密集型计算
- AI训练+推理混合负载
- 需要高MFU的大规模并行训练

不适合超节点的场景：
- 通信占比低的计算任务
- 间歇性/小规模任务
- 对成本极度敏感的场景

### 选型计算公式

**算力需求测算**：
```
有效算力 = 6 × 参数量 × 训练Token数 / 训练时间(秒)
硬件总算力 = 有效算力 / MFU
```

**显存需求测算**（BF16精度）：
```
静态显存 = 参数量 × (2B权重 + 2B梯度 + 8B Adam状态)
总显存 = 静态显存 × 2（含动态显存和冗余）
```

### TCO对比维度
1. 硬件采购成本
2. 电力成本（PUE影响）
3. 运维成本
4. 软件适配成本
5. 训练效率提升带来的时间价值
6. 资源利用率提升带来的规模效益

## 集群搭建流程

### 阶段一：基础设施准备
1. 机房承重复核与加固
2. 供电系统改造（列头柜/小母线）
3. 液冷管路铺设（CDU+分集水器+环路管网）
4. 网络基础设施部署
5. 接地与防雷系统检查

### 阶段二：设备入场
1. 专用气垫货车运输
2. 卸货垂直搬运
3. 机柜就位到底座
4. 并柜固定

### 阶段三：连接调试
1. 供电连接（工业连接器对接）
2. 液冷管路连接（Manifold法兰对接）
3. 网络连接（Scale-up + Scale-out）
4. 漏液检测绳连接
5. 接地连接

### 阶段四：系统上线
1. CDU启动与管路排气
2. 供电上电（先控制电后动力电）
3. 管理网络初始化
4. 固件/驱动版本检查与升级
5. 训前一键深度健康巡检
6. 网络性能压测
7. 业务部署
