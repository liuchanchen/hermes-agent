# 中国新能源车落地价估算参考

## 2026年 上海 纯电车落地价公式

```
落地价 ≈ 指导价 + 保险(6k-8k) + 上牌杂费(150元)
```

### 各项费用明细

| 项目 | 状态 | 说明 |
|------|------|------|
| 国家补贴(国补) | ❌ 已结束 | 2022年12月31日终止 |
| 购置税 | ✅ 全免 | 纯电动车2026年继续免征购置税 |
| 车船税 | ✅ 全免 | 纯电动乘用车免征车船税 |
| 上海绿牌 | ✅ 免费 | 价值约9-10万元（免拍卖费） |
| 交强险 | 950元 | 固定费用 |
| 商业险 | ~5,500-7,000元 | 视保额和折扣 |
| 上牌费 | ~150元 | 牌照工本费+行驶证 |

### 示例

**指导价27万的纯电车在上海落地：**
- 落地价 ≈ 270,000 + 7,000(保险) + 150(上牌) = **277,150元**

**注意**：此计算适用于CLTC续航≥200km、能量密度≥125Wh/kg的纯电动车。

## 2026年数据政策参考 (2026年5月查询)

- 财政部/工信部公告：新能源车购置税减免延长（纯电继续全免）
- 国补已终止：财政部2022年通知确认
- 上海绿牌政策：2026年纯电继续免拍牌
